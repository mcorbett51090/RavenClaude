## Method & caveats

- **Coverage:** all 139 plugins, 26 clusters, one research agent per cluster (web search, high bar — empty was the expected outcome for slow-moving domains).
- **Panels** were single-agent multi-seat deliberations (3 named seats each) returning per-seat votes + a verdict; documented in the run transcript.
- **Low panel attrition** (usefulness passed 78/83; detail passed 100%) means the panels did *not* do the final narrowing — human-style curation + first-hand re-verification did. Treat the 69-item backlog as leads, not a merge queue.
- Full machine-readable run output (all 78 approved findings with each panel's reasoning + recommended edits) is in the workflow transcript for this session.
