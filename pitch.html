A whole expert team, inside Claude Code.

RavenClaude adds specialist AI teammates — for engineering, cloud, data, finance, security, design, and dozens more fields — to Claude Code. One lead coordinates them, and you decide what they can do on their own with point-and-click controls. No hiring, no build step.

What you get

Specialists on call

Architects, code reviewers, security auditors, data engineers, FP&A analysts and more — drop in the fields you need, and they come with you to every project.

Many fields, one standard

Engineering, cloud, data, finance, healthcare, design, and dozens more — every team follows the same review bar and hand-off rules, so quality doesn't drift between them.

You set the limits

Decide what agents can do on their own — deny, ask, or allow — per tool, with a click. A built-in review step checks risky commands before they ever run.

No build step

The portal, the dashboard, and this page are self-contained HTML — open them in a browser, no server and no build. Updates are a git pull.

Install in four steps

/plugin marketplace add <this-repo>
Register the marketplace once.
/plugin install ravenclaude-core@ravenclaude
Install the core (then any specialist plugins you want).
/reload-plugins
Load the agents, skills, and hooks into your session.
/init-agent-ready
Scaffold AGENTS.md, CLAUDE.md, and the layout guardrails for your repo.
