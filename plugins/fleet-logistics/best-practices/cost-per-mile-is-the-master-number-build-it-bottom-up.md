# Cost-per-mile is the master number — build it bottom-up

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any fleet & logistics deliverable where this question is in play.

## The rule
All-in CPM ran ~$2.26 in 2024, with non-fuel marginal costs at a record ~$1.78. Build CPM from fixed + variable components; a single blended number hides where the cost lives.

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a fleet & logistics analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
