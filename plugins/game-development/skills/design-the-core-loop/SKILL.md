# Skill: Design the core loop

Features layered on a weak loop don't save it.

## Step 1 — Define the loop
The moment-to-moment action and its reward (§3 #2).

## Step 2 — Add the session loop
What brings the player back this session and next.

## Step 3 — Layer the meta
Progression and meta on top of a proven loop.

## Step 4 — Test for retention
Validate the loop holds attention before scaling.

## Output
A core-loop and session-loop design with the retention hooks and a meta layer.
