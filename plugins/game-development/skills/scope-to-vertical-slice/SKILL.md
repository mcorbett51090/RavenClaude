# Skill: Scope to a vertical slice

Scaling content before the loop is fun builds expensive games nobody plays.

## Step 1 — Define the slice
The minimum build that proves the core loop end-to-end (§3 #1).

## Step 2 — Sequence the risk
Put the riskiest unknowns (fun, tech) first (§3 #3).

## Step 3 — Set the fun bar
Define what 'the loop is fun' means to pass the gate.

## Step 4 — Gate the full build
Only greenlight content scaling after the slice passes.

## Output
A vertical-slice scope, a risk sequence, a fun bar, and a build gate.
