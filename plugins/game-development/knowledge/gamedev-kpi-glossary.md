# Game-development KPI glossary

The metrics a game is judged on — formulas, the misreads, and where a figure needs a `[unverified]` mark.

## Retention metrics

D1/D7/D30 retention, session length/frequency, churn. Retention is the first gate (§3 #4).

## Monetization metrics

ARPDAU, ARPPU, conversion rate, LTV — gated on retention (§3 #4).

## Production metrics

Milestone burn-down, risk burn-down, content cost-per-hour, scope variance (§3 #3, #6).

## Sourcing note

Figures in this file are from the author's domain knowledge and are marked `[unverified — training knowledge]` or `[ESTIMATE]` at point of use. Validate against a primary source before putting any figure in a client deliverable (§3 cite-or-mark rule).
