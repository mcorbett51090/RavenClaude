# Prove the fun in a vertical slice before the full build

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any game development deliverable where this question is in play.

## The rule
A vertical slice that proves the core loop is the cheapest way to de-risk a game; scaling content before the loop is fun is how studios build expensive games nobody plays. [unverified — training knowledge]

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a game development analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
