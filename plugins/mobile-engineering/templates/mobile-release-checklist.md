# Mobile release checklist

- [ ] Code signing / provisioning automated in CI (devops-cicd)
- [ ] Phased/staged rollout configured
- [ ] Store review guidelines checked (permissions justified, privacy labels)
- [ ] Old client versions still work against the API (versioning)
- [ ] Crash/ANR reporting wired
- [ ] Push permissions + deep links tested on device
- [ ] Battery/network profile sane (no polling where push fits)
