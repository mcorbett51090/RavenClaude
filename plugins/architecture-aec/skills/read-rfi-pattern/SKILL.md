# Skill: Read the RFI/change pattern

RFIs trace to document coordination quality.

## Step 1 — Categorize the RFIs
By cause: coordination gap, ambiguity, or field condition (§3 #3).

## Step 2 — Find the pattern
Where the set is generating questions.

## Step 3 — Cost the rework
The CA and change-order cost of the pattern.

## Step 4 — Improve the next set
Feed the lessons into the next project's QC.

## Output
An RFI categorization, the pattern, the rework cost, and a next-set QC improvement.
