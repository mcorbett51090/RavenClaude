# AWS FinOps checklist

- [ ] Cost allocation tags on everything (owner/env/cost-center)
- [ ] Budgets + anomaly detection wired
- [ ] Rightsized on utilization data BEFORE any commitment
- [ ] Savings Plans/RIs for steady-state baseline only
- [ ] Backups automated AND restore tested
- [ ] Zombie sweep (unattached EBS, idle LBs, old snapshots, stale dev envs)
