# Docs structure (Diataxis)

| Section | Kind | Audience | Purpose |
|---|---|---|---|
| Getting started | Tutorial | newcomer | first success, hand-held |
| Guides | How-to | practitioner | accomplish specific tasks |
| Reference | Reference | integrator | look up exact facts (spec-driven) |
| Concepts | Explanation | anyone | understand the why / trade-offs |

_Keep the four separate; cross-link between them._
