# After-tax return is the return that matters

Asset location (place tax-inefficient assets in tax-deferred accounts, tax-efficient ones in taxable, growth in Roth) and tax-loss harvesting are near-free return the client keeps. But the wash-sale rule (a loss is disallowed if a substantially identical security is repurchased within the window) and the client's bracket gate the specifics — flag them and route the actual tax conclusion to a CPA. Getting the right asset in the right account type is often worth more than a marginally cheaper fund.
