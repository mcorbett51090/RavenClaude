# higher-education-administration — scenarios bank

Field-tested scenarios for the `higher-education-administration` specialists. Each file is one
real-shaped situation: the problem, the context, the attempts (with outcomes), and the resolution —
ending with the action for the next consultant who hits the pattern.

These are teaching cases, not rules. The rules live in [`../best-practices/`](../best-practices/);
the decision trees live in [`../knowledge/`](../knowledge/).

> Benchmark figures are institution-, sector-, and date-dependent — treat as `[unverified]` and
> validate against the institution's own data before any deliverable.
