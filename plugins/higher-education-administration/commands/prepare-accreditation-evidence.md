# /prepare-accreditation-evidence

Spawn `academic-operations-and-compliance-coordinator` (with the IR analyst for the data artifacts) to
build an accreditation-evidence system.

## What it does

1. Maps each accreditation standard to the artifact(s) that demonstrate it.
2. Designs an outcomes-assessment loop that **closes** (define → measure → analyze → act → re-measure).
3. Audits for the highest-risk gaps (thin/stale evidence, loops that never act).
4. Coordinates a FERPA-aware data flow for student-level evidence.

## Usage

```
/prepare-accreditation-evidence
```

Then share your accreditor and current evidence/assessment state. The agent applies
[`accreditation-evidence-and-assessment`](../skills/accreditation-evidence-and-assessment/SKILL.md)
and the [`accreditation-evidence-map`](../templates/accreditation-evidence-map.md) template.

> Decision-support only — map to your accreditor's **current** standards and verify with institutional
> counsel; standards vary by accreditor and change on review cycles.
