You are running `/event-management:event-debrief`. Use `event-marketing-revenue` + the `post-event-measurement` skill.

## Steps
1. Restate the goal and KPIs that were set up front — measure against those, not a flattering substitute.
2. Compute ROI as outcome ÷ fully-loaded cost; report results vs each KPI target.
3. Lay out the registration funnel actuals (reach → register → attend) and the no-show rate.
4. Compare budget actual vs plan, including where contingency was spent.
5. Capture what broke / what the plan-B caught, sponsor fulfillment + renewal signal, and 3-5 carry-forward lessons.
6. Emit using `templates/post-event-report.md` + the Structured Output block.
