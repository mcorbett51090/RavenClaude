You are running `/event-management:plan-event`. Use `event-strategist` + the `design-event-plan-and-budget` skill.

## Steps
1. Name the primary goal (revenue / pipeline / community / education) and 2-4 KPIs with targets + baseline.
2. Traverse the format and break-even trees in `knowledge/event-management-decision-trees.md`.
3. Build the budget with a named contingency line; compute break-even.
4. Define the sponsorship strategy (tiers + what each delivers).
5. Set the go/no-go gate(s) with a date and a hard threshold, placed before non-refundable spend.
6. Emit using `templates/event-plan-and-budget.md` + the Structured Output block.
