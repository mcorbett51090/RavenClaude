# Pseudonymization is not anonymization

Pseudonymized data — re-identifiable by combining it with a separately-held key — remains personal data and stays fully in scope of privacy law; true anonymization, where re-identification is no longer reasonably possible, is a high bar that most 'anonymized' datasets do not actually meet. Conflating the two leads teams to under-protect personal data they wrongly believe is exempt. Name which one you have actually achieved.
