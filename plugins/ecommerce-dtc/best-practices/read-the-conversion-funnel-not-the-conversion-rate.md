# Read the conversion funnel, not the conversion rate

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any e-commerce & dtc deliverable where this question is in play.

## The rule
Average stores convert ~1.4–1.8% (good ~2.5–3%, top ~4.7%); a low rate is a traffic-quality, product-page, or checkout problem — diagnose the stage.

## Why
This is house opinion §3 #4. Getting it wrong is the most common way a e-commerce & dtc analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
