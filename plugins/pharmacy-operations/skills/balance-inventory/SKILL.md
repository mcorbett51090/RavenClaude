# Skill: Balance inventory and days-on-hand

Too much inventory is dead cash; too little is a stockout and a lost script (§3 #2).

## Step 1 — Measure days-on-hand
Inventory value ÷ daily COGS, by drug class.

## Step 2 — Separate specialty
Specialty/refrigerated/340B handled distinctly — large tied-up cash on specialty (§3 #2 #6).

## Step 3 — Read the balance
Tied-up cash vs stockout risk per class (§3 #2).

## Step 4 — Set class-specific targets
Not a blanket DOH target across the book (§3 #2 #6).

## Output
A days-on-hand read by class with specialty handled distinctly and the cash/stockout balance named.
