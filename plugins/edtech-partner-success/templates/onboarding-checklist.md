# Onboarding Checklist — First 90 Days — `<Partner Name>`

**Contract close date:** `<YYYY-MM-DD>`
**Go-live target:** `<YYYY-MM-DD>`
**PSM owner:** `<name>`
**Named partner-side champion:** `<name + role>`
**Segment:** `<K-12 / higher-ed / corporate L&D>`

---

## Week 0 (immediately post-close)

- [ ] Partner profile started (route to `partner-profile-curator`)
  - [ ] Institutional history captured
  - [ ] Decision-makers + stakeholders mapped
  - [ ] Named programs / partner terminology captured
- [ ] Welcome touchpoint with named champion
- [ ] Kickoff call scheduled (within 7 business days)
- [ ] Success plan drafted (route to `edtech-partner-success-manager` + skill `success-plan-authoring`)
- [ ] Baseline measurement plan agreed (route to `learning-analytics-analyst`)

---

## Days 1–30

- [ ] Kickoff call completed; partner-stated goals captured verbatim
- [ ] Success plan signed off (in writing) by named decision-maker
- [ ] Rostering / SIS / LMS integration validated (route to `rostering-data-quality` skill if any issue)
- [ ] Baseline measurements captured for all outcomes in success plan
- [ ] First touchpoint cadence locked (segment-aware)
- [ ] Health-score dashboard configured for this partner
- [ ] Champion-redundancy assessed (1 / 2 / 3+) — if 1, action plan to grow
- [ ] First small-win identified and surfaced to partner

---

## Days 31–60

- [ ] Mid-cycle check-in completed
- [ ] First measurable outcome trend visible (even if not yet at target)
- [ ] Rostering data quality re-validated (route to `rostering-data-quality`)
- [ ] Initial health score reviewed; weights tuned if needed
- [ ] Champion engagement confirmed; second champion candidate identified
- [ ] Any partner-side admin / configuration gaps closed
- [ ] First touchpoint-log review by PSM lead (process check)

---

## Days 61–90

- [ ] First measurable outcome assessed against target
- [ ] Refresh decision documented (continue / adjust / escalate)
- [ ] QBR drafted (route to `qbr-composer` + skill `qbr-composition`)
- [ ] Partner profile refreshed; "partner says" and "we observe" sections current
- [ ] "What we promised" section current
- [ ] Health-score signals validated against actual partner behavior
- [ ] Year-1 plan drafted (the 90-day plan + quarterly plans for the rest of year 1)

---

## Risks to watch during onboarding

- [ ] **Champion single-point-of-failure** — if redundancy = 1, growing the bench is a 90-day priority
- [ ] **Rostering / SIS / LMS sync gaps** — almost always surface in days 30–60; address immediately
- [ ] **Partner-side calendar collision** — start-of-year (K-12), finals week (higher-ed), Q4 close (corporate L&D) can stall onboarding; plan around
- [ ] **Measurement gaps** — if baselines can't be captured cleanly, the 90-day outcome assessment will be inconclusive; instrument before launch
- [ ] **Verbal-only agreements** — anything important must be in writing (success plan sign-off, commitments captured in touchpoint log)

---

## Handoff readiness check (end of day 90)

- [ ] Partner profile complete and up-to-date
- [ ] Touchpoint log has substantive entries (not "synced with X")
- [ ] Health score is configured, reviewed, and predictive
- [ ] Success plan is signed off and being executed
- [ ] First QBR is scheduled or completed
- [ ] Any red-flag triggers have been tested in practice
- [ ] PSM lead has reviewed the partner's state

If all checks pass, the partner is ready for steady-state PSM cadence. If not, document the gaps and a plan to close them in days 91–120.
