## Scorecard Dimensions and Weights

| Dimension | Weight | Pass | Partial | Fail |
| --- | --- | --- | --- | --- |
| Resolution accuracy | 40% | Issue fully resolved; customer confirmed resolution or resolution is verifiable | Issue partially resolved; follow-up required through no agent fault | Issue unresolved; wrong answer given or issue closed without resolution |
| Communication quality | 25% | Clear, grammatically correct, professional tone throughout; no ambiguity | Minor clarity issues; one or two grammatical errors; tone appropriate | Confusing or unprofessional response; customer would struggle to understand next step |
| Empathy and rapport | 15% | Acknowledges customer situation; de-escalates if frustrated; personalizes beyond template | Acknowledges but minimally; no de-escalation attempt when warranted | Dismissive, robotic, or escalating tone; no acknowledgment of customer frustration |
| Process adherence | 10% | Correct macro/template used; ticket tagged correctly; escalation protocol followed | One minor deviation; e.g., tag missing but macro correct | Macro bypassed incorrectly; wrong escalation path; required fields left blank |
| Escalation discipline | 10% | Escalation necessary and reason code recorded; context fully documented for receiving tier | Escalation borderline; reason code present but brief | Unnecessary escalation; or required escalation missed; reason code absent |

**Scoring formula:**
`Total score = Σ (dimension weight × dimension score)`, where Pass = 1.0, Partial = 0.5, Fail = 0.

## Calibration Protocol

**Anchor conversations:** Select [3–5] conversations per review period — one clear Pass, one clear
Fail, one ambiguous. All reviewers score independently before discussing.

**Variance threshold:** No more than [10] percentage points difference in weighted total score
between any two reviewers. Variance above this threshold → calibration meeting required.

**Calibration meeting agenda:**
1. Share individual scores (no discussion yet).
2. Identify dimensions with highest variance.
3. Review rubric language for that dimension together.
4. Reach consensus via rubric, not seniority.
5. Log the outcome; update rubric language if the anchor case exposed an ambiguity.

**Calibration frequency:** [Monthly / per review cycle]

## Coaching Loop (reference)

1. Share the scored conversation with the agent within [48 hours] of review.
2. Name the specific behavioral gap (not a personal judgment).
3. Agree on one behavioral change for the next [2-week] period.
4. Pull [3] follow-up conversations from the same category for a re-review at the end of the period.
5. If no improvement: escalate to manager + coaching plan. Log in QA tracker.
