# Changelog — embedded-iot-engineering

All notable changes to this plugin are documented here. Versioning is semver; bump on every user-visible change (AGENTS.md).

## [0.1.0] — 2026-06-08

Initial release.

- **4 agents** — `embedded-systems-lead`, `firmware-rtos-specialist`, `power-budget-analyst`, `connectivity-protocol-specialist`, each carrying the full scenario-authoring schema.
- **5 skills + 5 commands** — `build-power-budget`, `verify-real-time`, `budget-memory`, `select-protocol`, `plan-ota`.
- **4-file knowledge bank** — KPI glossary, unit economics, 2025–2026 context, and Mermaid decision trees.
- **`scripts/embedded_iot_calc.py`** — stdlib calculator: `power-budget`, `memory-budget`, `bom-cost`. Decision-support only.
- **4 templates · 8 best-practice rules · scenarios bank · 1 advisory hook**.
