# Containment is not a CAPA

Scrapping or reworking the bad part is containment — it stops the immediate bleeding but fixes nothing. A nonconformance closed with containment alone is scheduled to recur. A real CAPA separates the corrective action (fix this batch) from the preventive action (remove the root cause so it can't happen again), verifies effectiveness, and updates the control plan / FMEA. Closing an NCR with no root-cause analysis and no preventive action is closing nothing.
