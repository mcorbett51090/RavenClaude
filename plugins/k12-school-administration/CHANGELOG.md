# Changelog — k12-school-administration

All notable changes to this plugin are documented here. Versioning is semver; bump on every user-visible change (AGENTS.md).

## [0.1.0] — 2026-06-08

Initial release.

- **4 agents** — `school-administration-lead`, `enrollment-attendance-analyst`, `staffing-budget-specialist`, `student-outcomes-specialist`, each carrying the full scenario-authoring schema.
- **5 skills + 5 commands** — `model-enrollment-funding`, `flag-chronic-absenteeism`, `fit-staffing-to-budget`, `allocate-per-pupil`, `read-outcomes-segmented`.
- **4-file knowledge bank** — KPI glossary, unit economics, 2025–2026 context, and Mermaid decision trees.
- **`scripts/k12_school_administration_calc.py`** — stdlib calculator: `enrollment-funding`, `staffing-ratio`, `absenteeism`. Decision-support only.
- **4 templates · 8 best-practice rules · scenarios bank · 1 advisory hook**.
