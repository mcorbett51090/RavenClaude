# The PART exam is the backbone of a defensible note

**Rule.** Build the necessity documentation on PART — **P**ain/tenderness, **A**symmetry, **R**ange-of-motion
abnormality, **T**issue/tone changes — documenting the findings (commonly ≥2 per the Medicare framework
`[verify-at-use]`) for the region coded, alongside the functional goal and progress.

**Why.** PART is the widely-recognized framework payers expect for chiropractic medical necessity; a note
organized around it maps directly to what an auditor looks for and to the region-based CMT code.

**Anti-pattern it kills.** A note with a diagnosis and a code but no exam findings to support that care
was needed for that region on that day.

**In practice.** PART findings → region coded → functional goal → progress since last visit. Verify the
specific payer's required findings/frequency — it's `[verify-at-use]`.
