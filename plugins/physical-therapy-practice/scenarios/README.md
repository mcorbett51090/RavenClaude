# physical-therapy-practice — scenarios bank

Field-tested scenarios for the `physical-therapy-practice` specialists. Each file is one real-shaped
situation: the problem, the context, the attempts (with outcomes), and the resolution — ending with
the action for the next consultant who hits the pattern.

These are teaching cases, not rules. The rules live in [`../best-practices/`](../best-practices/);
the decision trees live in [`../knowledge/`](../knowledge/).

> This plugin is operational decision-support, not coding/clinical/legal/compliance advice. Verify
> Medicare/CPT/payer specifics against current policy and a certified coder/compliance professional;
> use de-identified data.
