# Skill: Value storage by dispatch

Storage value is in the dispatch, not the raw kWh.

## Step 1 — Define the use-case
Arbitrage, demand-charge reduction, backup, or capacity (§3 #7).

## Step 2 — Model the dispatch
Value the battery's operation against the tariff/market.

## Step 3 — Net against cost
Compare the dispatch value to the storage cost and incentive.

## Step 4 — Recommend the sizing
Size the battery to the highest-value dispatch.

## Output
A use-case, a dispatch-value model, a cost comparison, and a sizing recommendation.
