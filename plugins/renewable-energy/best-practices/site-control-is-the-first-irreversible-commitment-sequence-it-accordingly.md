# Site Control Is the First Irreversible Commitment — Sequence It Accordingly

**Status:** Absolute rule
**Domain:** Project development / development timeline
**Applies to:** `renewable-energy`

---

## Why this exists

Solar project development has a well-defined sequencing problem: site control (lease or purchase option) must be obtained early enough to start the interconnection application, but is also the first commitment that carries real cost and contractual risk. A developer who sequences site control before preliminary feasibility (resource screening, interconnection queue position, permitting viability) risks paying for and holding land on a project that will never achieve NTP. A developer who holds off on site control until full feasibility is complete risks losing the site to a competitor. The correct sequence is: preliminary screen first, then option (not lease), then full feasibility, then exercise.

## How to apply

Follow the three-phase site control sequence:

```
Site control sequencing protocol:
Phase 1 — Preliminary screen (before any payment):
  [ ] Satellite and resource screening (NREL NSRDB, PVWatts): viable solar resource?
  [ ] Interconnection queue screening: is there queue capacity in the relevant substation footprint?
  [ ] Zoning check: is solar a permitted use or conditional use (not prohibited)?
  [ ] Setback and environmental flags: known wetlands, floodplain, habitat?
  → Go/No-go gate: all four items must clear before any site control cost.

Phase 2 — Option agreement (not lease):
  [ ] Execute a land option (right to lease, not a lease): 12–24 months, renewable
  [ ] Option payment: $500–$2,000/acre typical [unverified — training knowledge]
  [ ] Option must survive assignment (transfers with project if sold)
  [ ] Begin interconnection pre-application screening while option is active

Phase 3 — Exercise (only after key feasibility gates clear):
  [ ] Interconnection pre-application results indicate feasible path
  [ ] Permitting jurisdiction confirms conditional use permit eligibility
  [ ] Exercise option to lease or purchase at agreed terms
```

**Do:**
- Structure the option to be long enough to reach a COD milestone in the interconnection queue — a 12-month option may expire before study results arrive.
- Include a memorandum of lease or option in the county recorder's office to protect against the landowner selling the property to a third party during the option period.
- Negotiate an exclusivity clause preventing the landowner from signing with a competing developer during the option term.

**Don't:**
- Execute a full lease before clearing the interconnection queue pre-application; the project may not be feasible and the lease obligation remains.
- Allow the option to expire while waiting for interconnection study results; extend proactively with 90 days of runway remaining.

## Edge cases / when the rule does NOT apply

Brownfield or parking lot canopy projects (carports, rooftop) where the "site" is a long-term offtake customer's owned property have a different control structure — the PPA or lease-back arrangement IS the site control. Community solar projects may use a different land control mechanism depending on state community solar statute.
