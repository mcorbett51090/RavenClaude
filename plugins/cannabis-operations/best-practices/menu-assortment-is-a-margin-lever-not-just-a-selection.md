# Treat menu assortment as a margin lever, not just a product selection

**Status:** Pattern
**Domain:** Cannabis operations / dispensary retail / category management
**Applies to:** `cannabis-operations`

---

## Why this exists

A dispensary menu is the primary lever available to the operator between reorders — but most operators build menus around vendor availability and consumer requests rather than margin contribution by SKU. The result is a menu heavy with low-margin flower (where the operator competes on price) and light on the category mix (pre-rolls, edibles, concentrates, accessories) that reliably delivers 50–70%+ gross margin. Under 280E, the operator cannot deduct selling expenses, so every margin point the menu leaves on the table is compounded by the tax burden. A margin-aware menu review is not optional for a healthy dispensary P&L.

## How to apply

Run a margin-contribution analysis quarterly:

```
SKU / category | Units sold | Revenue | COGS | Gross margin % | Contribution $
---------------|-----------|---------|------|----------------|----------------
Flower (bulk)  |           |         |      |                |
Pre-rolls      |           |         |      |                |
Edibles        |           |         |      |                |
Concentrates   |           |         |      |                |
Vapes          |           |         |      |                |
Accessories    |           |         |      |                |
```

Sort by contribution dollars descending. Flag any SKU where gross margin % is below the store target and contribution dollars are low — these are first candidates for delisting or price adjustment. Flag any high-margin category below its target shelf/menu space share.

**Do:**
- Set a minimum gross-margin floor per category before adding a new SKU.
- Review the menu for "velocity traps" — high-unit-count SKUs that drive traffic but deliver low margin contribution.
- Pilot new high-margin categories (accessories, topicals) even at low velocity before the category is mature.
- Tie menu rebalancing to a specific gross-margin-per-transaction target.

**Don't:**
- Remove high-velocity, low-margin SKUs without a transition plan — they may anchor foot traffic that converts to higher-margin items.
- Optimize purely on margin without watching basket mix — a menu with no entry-price SKUs will lose new customers.
- Let vendor relationships (co-op deals, exclusives) override margin discipline without a written exception.

## Edge cases / when the rule does NOT apply

- Medical-only dispensaries with restricted formularies may have limited assortment flexibility mandated by their license conditions.
- Operators in thin-competition markets may sustain a simpler menu longer — but the margin discipline still applies when competition increases.
