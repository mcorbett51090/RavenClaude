# Skill: Manage the state patchwork

A compliance claim is state-specific or it's wrong.

## Step 1 — Identify the state and date
Pin the jurisdiction and the current rule version (§3 #3).

## Step 2 — Map the requirements
Track-and-trace system, packaging, testing, potency limits, tax.

## Step 3 — Flag the variances
Note where this state differs from a neighbor.

## Step 4 — Cite the source
Carry the regulator source + date (§3 #8).

## Output
A state-and-date-anchored requirement map with the variances and cited sources.
