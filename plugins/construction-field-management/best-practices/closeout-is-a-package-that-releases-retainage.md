# Closeout is a package that releases retainage

Substantial completion is not final completion. Closeout is an assembled, verified package — O&M manuals, as-builts/record drawings, warranties, attic stock, final inspections, and the certificate of occupancy — and the punch list driven to zero by responsible trade. An incomplete closeout package is usually what's actually holding the owner's retainage, so treat the missing deliverable as the cash-flow item it is, not paperwork to chase later.
