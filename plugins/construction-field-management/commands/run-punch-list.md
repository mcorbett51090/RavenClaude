You are running `/construction-field-management:run-punch-list`. Use `field-and-safety-coordinator` + the `field-qaqc-and-safety` skill.

## Steps
1. Deduplicate the punch items; assign each a responsible trade, a location, and a target date with ball-in-court.
2. Separate substantial-completion blockers from final-completion items; name the gap and what each blocks.
3. Identify any item that is actually a non-conformance/backcharge (a sub's defect) and route the cost to `cost-and-change-controls-lead`.
4. Checklist the closeout package: O&M manuals, as-builts/record drawings, warranties, attic stock, final inspections, CO — and name what specifically is blocking retainage release.
5. Flag any items that need an inspection/hold-point sign-off or an AHJ/special inspection before close.
6. Emit the punch-to-closeout plan + the Structured Output block (with `Field/cost/schedule impact:` and `Ball-in-court:`). Route the acceptance spec to `architecture-aec`, the schedule impact to `project-management`.
