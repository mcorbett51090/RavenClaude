# The note is a legal record

Progress notes are documented contemporaneously, in a consistent format (DAP / SOAP / BIRP), factual and behavioral — not editorialized or written long after the fact. If it isn't documented, it didn't happen; if it's documented inconsistently across clinicians, it doesn't survive an audit. Standardize the *structure* so nothing required (diagnosis, intervention, response, plan) is missed, and leave room for the clinician's clinical voice within it. The standard governs the format; the clinician authors the content.
