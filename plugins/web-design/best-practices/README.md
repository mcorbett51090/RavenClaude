# Web-design best-practice docs

Named, citable rules for the `web-design` plugin's specialist agents and any consumer building a site with them. Each file is one rule — read, applied, and cited as a whole. Grounded in this plugin's `knowledge/` briefs and agent constitutions; the cross-marketplace index lives in [`../../../docs/best-practices/README.md`](../../../docs/best-practices/README.md).
