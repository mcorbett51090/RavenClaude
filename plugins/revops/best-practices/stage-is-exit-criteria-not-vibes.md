# A stage is exit criteria, not vibes

Every pipeline stage has an objective, verifiable exit criterion — a buyer action (demo booked, mutual action plan signed, procurement engaged), not a seller's hope. A stage defined by rep optimism is precisely why the forecast misses: the probability attached to "stage 3" is meaningless if stage 3 means something different for every rep. Write the exit criterion as something an outside observer could confirm, and compute the stage's conversion from your own history rather than the CRM's out-of-the-box defaults.
