# Quota is built bottoms-up from capacity, not top-down from the board number

Reconcile the board target against ramped-rep capacity × productivity before you set a quota; a number handed down from the board with no capacity model is a wish that misses predictably. Bottoms-up capacity = (fully-ramped reps + partially-ramped reps × their ramp fraction) × expected productivity per rep — and that product is the surface you reconcile against the board ask. When the two don't meet, the gap is a *staffing/ramp/productivity* decision (hire earlier, shorten ramp, raise productivity, or accept the gap), not a bigger number stapled onto the same heads. A quota set top-down only also breaks comp: reps assigned an un-makeable number disengage or game it, so the capacity reconciliation protects attainment behavior as much as the forecast. Build the capacity model first, state the ramp curve and productivity assumption explicitly, then negotiate the board number against it — never the reverse.
