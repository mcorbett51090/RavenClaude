# Standardize protocols to kill unwarranted variation

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any veterinary practice deliverable where this question is in play.

## The rule
The biggest quality and margin lever in a multi-DVM practice is reducing variation in how the same presentation is worked up and treated. A protocol is decision-support for the licensed DVM, never an order.

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a veterinary practice analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
