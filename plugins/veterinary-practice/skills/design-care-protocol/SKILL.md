# Skill: Design a care protocol

Standardizing the common workups is the biggest quality-and-margin lever in a multi-DVM practice.

## Step 1 — Scope the presentation
Define the presentation and the decision points (history, diagnostics, treatment).

## Step 2 — Align to evidence and stay decision-support
Build the protocol to current guidance, framed as support for the DVM, never an order (§3 #1).

## Step 3 — Standardize the estimate
Tie the protocol to a consistent estimate so recommended-care presentation is uniform (§3 #4).

## Step 4 — Plan adoption
Define how DVMs adopt and where deviation is expected and documented.

## Output
A protocol pack with decision points, a standard estimate, and an adoption plan — as DVM decision-support.
