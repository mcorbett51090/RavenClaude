# Production per hour is the capacity lens

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any dental practice deliverable where this question is in play.

## The rule
Dentist production runs ~$475–$575/hr (high performers $700+) and hygiene ~$145–$175/hr; per-hour production, not per-day, exposes the real capacity story.

## Why
This is house opinion §3 #4. Getting it wrong is the most common way a dental practice analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
