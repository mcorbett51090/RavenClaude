# Overhead is the master margin number

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any dental practice deliverable where this question is in play.

## The rule
National median overhead sits ~62% of collections; high performers run 55–60%. Read overhead against a benchmark before debating any single cost — wages alone are ~25–30% of collections.

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a dental practice analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
