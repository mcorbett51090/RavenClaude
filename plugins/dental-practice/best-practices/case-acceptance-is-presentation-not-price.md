# Case acceptance is presentation, not price

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any dental practice deliverable where this question is in play.

## The rule
Acceptance is built on how the treatment plan is presented and sequenced, not on discounting; a low acceptance rate is usually a communication problem.

## Why
This is house opinion §3 #3. Getting it wrong is the most common way a dental practice analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
