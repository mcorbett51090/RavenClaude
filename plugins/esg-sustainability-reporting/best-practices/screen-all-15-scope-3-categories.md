# Screen all 15 Scope-3 categories — exclusion is a documented decision

Scope 3 is not optional and it is not cherry-picked. Screen all 15 categories (upstream 1-8, downstream 9-15) for relevance; include every material one; and for any category excluded, record *why* — not relevant to the business, immaterial against a sourced threshold, or no reasonable estimation method available. An omitted material category with no rationale is a defect an assurer will find, and a Scope-3 boundary that quietly drops the hard categories (purchased goods & services, use of sold products, investments/financed emissions) overstates a "reduction" by shrinking the denominator. Document the screen as a table — category, relevant?, included?, method, data-quality tier — so the boundary itself is assurable, not just the totals inside it.
