# Scope before you calculate

The framework and the materiality determination decide *what* must be disclosed; the reporting boundary decides *for which entities*. Never build a GHG inventory or draft a disclosure before the applicable standard, the materiality test, and the consolidation/value-chain boundary are fixed. A precise inventory of the wrong scope is still wrong — calculating the wrong question accurately wastes the work and fails assurance. Fix the boundary first, then calculate inside it.
