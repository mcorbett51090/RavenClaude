# Data quality is tiered and disclosed, not implied

Every inventory line rests on data of a known quality: metered/primary supplier-specific data, secondary/proxy averages, or a spend-based estimate, with the gap between them often an order of magnitude. Tier each line explicitly (e.g. primary / secondary / proxy / spend-based), name the method, and disclose the mix — a single emissions total with no quality signal invites a reader to trust a spend-based estimate as if it were metered. The tier also drives improvement: it shows where primary data would most change the number and where an estimate is good enough. An assurer tests the weakest-tier lines first; a figure presented as precise that is actually a proxy is a finding. Disclose the tiering alongside the totals, and flag any material line resting on the lowest tier.
