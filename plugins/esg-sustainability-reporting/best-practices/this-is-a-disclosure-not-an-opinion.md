# This is a disclosure, not an opinion

The plugin scopes the report, calculates the inventory, drafts the disclosure, and readies it for assurance. It does **not** issue the legal-sufficiency opinion on a filing or the audit/assurance opinion on the figures — those belong to counsel and the assurance provider, and naming that seam is part of every output, not an optional footnote. The audited financial number a climate disclosure ties to is produced and audited by `finance`; the activity-data lineage is owned by `data-governance-privacy`; the filing mechanic is `regulatory-compliance` plus counsel. Crossing those seams — asserting a figure is "audited", a filing is "legally sufficient", or a disclosure "will pass assurance" — manufactures a false guarantee. State what the plugin produced, what bar it was built to, and who owns the opinion that has not yet been given.
