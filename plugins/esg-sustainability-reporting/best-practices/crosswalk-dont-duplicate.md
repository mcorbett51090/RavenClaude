# Crosswalk, don't duplicate

When multiple frameworks apply (e.g. CSRD/ESRS AND ISSB IFRS S1/S2, often plus GRI and the SEC rule), map the overlapping data points once and disclose against each. A single sourced figure should serve every framework it satisfies — running CSRD and ISSB as two separate projects duplicates effort, multiplies error surface, and invites inconsistent numbers across reports. Build the crosswalk early (ESRS ⇄ IFRS S ⇄ GRI ⇄ SEC), source each shared data point once, and let one number flow to every disclosure that needs it.
