# Materiality is a determination, not a vibe

Double materiality (impact AND financial — CSRD/ESRS) and financial materiality (ISSB/SEC) are different tests with different outputs; GRI runs impact materiality. Name which test you ran, the axes you assessed, the governance that signed off, and the evidence behind each material topic. A stakeholder survey is one input, not a determination. An assurer will ask how each material topic was determined — a materiality assessment with no named test, no governance, and no per-topic evidence will not survive that question.
