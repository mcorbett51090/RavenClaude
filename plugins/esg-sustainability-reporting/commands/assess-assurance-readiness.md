## Steps
1. Fix the target assurance level (limited vs reasonable) before drafting — they demand different evidence depth and controls.
2. Draft the disclosure against the cited framework clauses (ESRS / IFRS S1·S2 / GRI / SEC); each material disclosure names the requirement it satisfies.
3. Design the evidence trail into each figure (activity data, factor source/vintage, method, sign-off) and build testable data controls (completeness, accuracy, cutoff, consolidation).
4. Run a claim-substantiation review: map each claim to its reference; report gross emissions and offsets separately; flag any cherry-picked boundary; remove or rewrite unsupportable claims.
5. Run the gap assessment — map each disclosure to what the engagement will test, rank gaps by exposure, sequence remediation; assemble the auditor-liaison pack (evidence index, methodology memo, management assertions).
6. Route the seams: inventory gaps → `ghg-accounting-analyst`; data lineage → `data-governance-privacy`; the filing/opinion → `regulatory-compliance` / counsel / the assurance provider. Emit the Structured Output block (with `Framework & clause:` and `Assurance posture:`).
