## 1. First-30-days onboarding sequence
| Step | Timing | Owner | Success signal |
|---|---|---|---|
| Book first visit | At signup | Front desk | Visit attended within X days |
| Goal + baseline set | First visit | Coach | Goal recorded |
| Early-frequency target | Weeks 1–4 | Membership | Hits habit threshold _[verify-at-use]_ |
| Honeymoon check-ins | Scheduled cadence | Membership | Re-book after each |
| Sticky milestone reached | ~Day 30 | Membership | Crosses retention-predictive attendance |

## 2. Attendance-signal early-warning system
| Signal | Threshold | Risk | Action / owner |
|---|---|---|---|
| Visits/week trending down | _set_ | Rising | Proactive re-engage |
| Days-since-last-visit | _set_ | High | Personal outreach before cancel |
| Missed post-onboarding re-book | 1 miss | Early cliff | Onboarding follow-up |

## 3. Cause-first save flow
| Cancel cause | Save (not a discount) | Owner |
|---|---|---|
| Price / affordability | Downgrade tier or class-pack | Membership |
| Life change / travel | Freeze or pause | Membership |
| Injury / health | Medical pause + return plan | Coach + membership |
| Boredom / plateau | Program change / new class / PT assessment | Coach |
| Moved away | Graceful let-go + win-back list | Membership |
