# Read cost-to-raise-a-dollar by channel, not blended

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any nonprofit fundraising deliverable where this question is in play.

## The rule
Each channel (events, direct mail, grants, major gifts, digital) has a different cost ratio; a blended number hides which channel is subsidizing which.

## Why
This is house opinion §3 #4. Getting it wrong is the most common way a nonprofit fundraising analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
