# Segment donors by value, recency, and engagement

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any nonprofit fundraising deliverable where this question is in play.

## The rule
Treating a $25 first-timer and a $25k loyal donor the same wastes both; RFM-style segmentation drives where the cultivation hours go.

## Why
This is house opinion §3 #3. Getting it wrong is the most common way a nonprofit fundraising analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
