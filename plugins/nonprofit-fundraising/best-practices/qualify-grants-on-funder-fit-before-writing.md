# Qualify grants on funder fit before writing

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any nonprofit fundraising deliverable where this question is in play.

## The rule
A proposal to a poorly-fit funder is sunk cost; score alignment (mission, geography, grant size, history) before investing the writing hours.

## Why
This is house opinion §3 #2. Getting it wrong is the most common way a nonprofit fundraising analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
