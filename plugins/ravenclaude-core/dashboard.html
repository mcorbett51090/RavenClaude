RavenClaude comfort posture

Categorical permission-autonomy scale for Claude Code. Edited via this plugin's dashboard.html Settings tab; saved to .ravenclaude/comfort-posture.yaml in the consumer's project. The /set-posture skill (proposal 002 §5) translates this YAML into .claude/settings.json permission rules.

ravenclaude-core · static dashboard, no backend. Edits stay in your browser until you click Download.

Apply a preset

Sets every category at once. You can still override individual rows below.

Global default
Used for any category not explicitly set below. Apply a preset to set this and all categories at once.
Files
Edit files outside this project
Writing to your home directory or system paths. Almost always wants to be cautious.
Edit files in this project
Write, Edit, MultiEdit on files inside the current project. Set lower if you want to review every change.
Read files outside this project
Reading ~/.ssh, ~/.aws, ~/.config, /etc, etc. Almost always wants to be cautious.
Read files in this project
ls, cat, grep, reading source files inside the current project directory.
Network
Web reads
WebFetch on known documentation domains. Productive means trusted domains never ask.
Web writes
POST/PUT/DELETE to web endpoints. Almost always wants to be cautious.
Other
MCP server tools
Tools provided by connected MCP servers. Per-server trust lives in your Claude Code config; this is the global default.
Shell
Arbitrary code execution
python -c, node -e, bash -c, eval. Recommend cautious — these are general-purpose.
Local mutations
mkdir, mv, cp, git commit. Changes your filesystem but doesn't talk to a server.
Package installs
npm install, pip install, apt install, brew install. Recommend cautious — these can run install scripts.
Read-only shell commands
ls, cat, git status, grep, find. The cheap stuff. Productive means "never ask."
Remote mutations
git push, gh pr create, gh pr comment. Anything that publishes to a shared system.
