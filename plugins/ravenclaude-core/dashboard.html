RavenClaude comfort posture

Categorical permission-autonomy scale for Claude Code. Edited via this plugin's dashboard.html Settings tab; saved to .ravenclaude/comfort-posture.yaml in the consumer's project. The /set-posture skill translates this YAML into .claude/settings.json permission rules. v4 adds top-level security_deny (always-on baseline) and per-pattern overrides inside each category.
