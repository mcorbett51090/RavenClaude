## Conventions

- **Interest:** how much they care about the project — High / Med / Low.
- **Influence:** how much they can affect outcomes — High / Med / Low.

The interest × influence quadrant drives engagement strategy:

| | High Influence | Low Influence |
|---|---|---|
| **High Interest** | **Manage closely** — weekly minimum touchpoint, full status reports | **Keep informed** — targeted updates, decision-needed prompts |
| **Low Interest** | **Keep satisfied** — concise periodic updates, no surprises | **Monitor** — ad hoc, on request only |

- **Cadence:** Weekly / Bi-weekly / Monthly / Ad hoc / On-demand.
- **Channel:** how they actually want updates — email, Teams, Slack, scheduled call, status portal, etc. Match their preferred channel; don't impose yours.
- **Update this register** when stakeholders change — new sponsor, new SME, role change, contact going on leave. The PM agent will re-tailor status report tone based on what's here.
