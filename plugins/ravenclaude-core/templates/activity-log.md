## Conventions

- **One section per day**, newest at top.
- **2–5 bullets per day.** If you find yourself writing a paragraph, it belongs in the status report or the RAID log, not here.
- **Cross-reference RAID items** by ID when relevant (e.g. *"Closed I-3 — flow now handles embedded commas."*).
- **No backfilling** more than 2 days. If you missed several days, do a status-report-style summary instead, then resume daily entries from today.
