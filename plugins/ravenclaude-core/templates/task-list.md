## Conventions

- **Priority:**
  - **P1** — must-do this week
  - **P2** — next 2 weeks
  - **P3** — someday / nice-to-have
- **Status:** Not started, In progress, Blocked, Done.
- **Stale flag:** any non-Done task with `Last update` >7 days ago triggers a review prompt from the project-manager agent.
- **Owner:** exactly one named person. No *"the team,"* no *"TBD."* If you don't know yet, the task isn't ready to be Active — leave it in Backlog.
- **Cross-references:** when a task ties to a RAID item, mention it in `Notes` (e.g. *"Mitigates R-3"*).
