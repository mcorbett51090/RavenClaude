# Surcharge glossary — who typically pays, and overlaps

> On-demand resource for the `freight-pricing-mechanics` skill. Who actually pays each line is ultimately set by the **Incoterm** (see the `incoterms-2020` skill); "typically" below is the common default, not a rule. All amounts are lane- and carrier-specific — confirm against your live tariff.

| Code | Full name | Basis | Typically billed to | Notes / overlap |
|---|---|---|---|---|
| BAF | Bunker Adjustment Factor | per container / per W-M / per kg | the freight payer (per Incoterm) | Fuel volatility. May be folded into an all-in. Overlaps LSS. |
| LSS | Low-Sulphur Surcharge | per container / per W-M | freight payer | IMO 2020 low-sulphur fuel. Sometimes replaces part of BAF — don't double-count. |
| CAF | Currency Adjustment Factor | % of base freight | freight payer | FX risk, often ~2–8%; spikes in currency crises. |
| OTHC | Origin Terminal Handling Charge | per container | seller on C/D-terms, buyer on F-terms | Separate from DTHC. |
| DTHC | Destination Terminal Handling Charge | per container | buyer on FOB/CFR/CIF; seller on DAP/DDP | The classic "who pays destination THC" dispute — resolve by Incoterm. |
| GRI | General Rate Increase | per container | freight payer | Carrier-announced, effective a date — validity-sensitive. |
| PSS | Peak Season Surcharge | per container | freight payer | Seasonal demand peaks. |
| ISPS / SEC | Security (ISPS Code) | per container / shipment | freight payer | Port security. Air equivalent: SSC (security surcharge). |
| AMS | Automated Manifest System (US) | per B/L | freight payer | US advance filing. |
| ISF | Importer Security Filing (10+2, US) | per shipment | importer | Penalties for late/incorrect filing. |
| ENS / ICS2 | Entry Summary Declaration (EU) | per B/L | freight payer | EU advance cargo data. |
| DDC | Destination Delivery Charge | per container | consignee | Mainly US destinations. |
| DOC | Documentation fee | per B/L / AWB | freight payer | B/L issuance. |
| Telex / Seaway | Telex release / seaway B/L | per B/L | freight payer | Speeds cargo release vs original B/L. |
| Demurrage | — | per container per day | whoever holds the box at port | Box sitting at terminal beyond free time. Distinct from detention. |
| Detention | — | per container per day | whoever holds the box | Box held outside the terminal (at the warehouse) beyond free time. |
| FSC | Fuel surcharge (air) | per kg | freight payer | Air analogue of BAF. |
| SSC | Security surcharge (air) | per kg | freight payer | Air screening/security. |
| VGM | Verified Gross Mass admin | per container | shipper | SOLAS weighing requirement (the weight itself is the shipper's duty). |

**Demurrage vs detention** is the most-confused pair: *demurrage* = container still **inside** the terminal past free time; *detention* = container **outside** the terminal (with the shipper/consignee) past free time. Use the right one in a quote or a dispute.
