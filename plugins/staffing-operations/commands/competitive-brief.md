# Build a competitive brief

You are running `/staffing-operations:competitive-brief` for `$ARGUMENTS`. Build it the way the `workforce-market-analyst` does — segment-resolved, current, sourced.

## Steps
1. **Name the segments** — score competitiveness within each; never average across them (§3 #9).
2. **Place the firm + competitors on the grid** using [`../knowledge/competitor-landscape.md`](../knowledge/competitor-landscape.md).
3. **Identify structural advantages** — what the firm's shape gives it (dual-segment breadth, allied + school-therapy depth, all-50-state, certifications).
4. **Name where competitors lead, by lane** — travel scale (Aya/AMN), locums (CHG/Jackson), teletherapy software (Presence/eLuma), subs (ESS/Kelly), therapy roll-up (TSSG), public capital (AMN/CCRN/Kelly).
5. **Verify currency** — Soliant's parent is **Vistria** (not Adecco); the **Aya–Cross Country merger was terminated** (Dec 2025). A stale fact is a credibility hit.
6. **Translate to where-to-play** — Defend / Invest / Cede per segment.

## Output
Fill [`../templates/competitive-landscape-brief.md`](../templates/competitive-landscape-brief.md). Pair with `/staffing-operations:market-trend-readout` for the trend context.

## Guardrails
- Every size/rank/ownership fact gets a URL + retrieval date; aggregator figures marked `[ESTIMATE]`/`[unverified]`.
- Don't claim a strength the firm doesn't have — strong in allied ≠ strong in travel nursing.
