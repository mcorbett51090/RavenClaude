# Product Metrics

## North Star + inputs
One metric = value customers get; decompose into **3-5 input metrics** a team can move. Signups-while-ignoring-retention is theater.

## Vanity vs actionable
Cumulative totals/raw counts flatter; **rates, cohorts, retention** reveal. Drop metrics you can't act on.

## Guardrails
Name what you must not harm (don't grow signups by tanking conversion).

## Judge outcomes
Did the target metric move vs **baseline/control**? Significance -> `applied-statistics`. A ship that moved nothing is a learning — say so.
