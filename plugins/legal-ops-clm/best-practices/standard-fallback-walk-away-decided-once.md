# Standard / fallback / walk-away, decided once

A clause library encodes, for each key clause, the standard (preferred) position, the fallback (acceptable) position, and the walk-away (never-acceptable) line — set by a lawyer once and applied consistently by everyone. The fallback is what lets a non-lawyer negotiate within bounds; without it every deviation escalates and the library is just a template. The walk-away line is real — name it so a deal can't drift past the point a lawyer would have stopped it.
