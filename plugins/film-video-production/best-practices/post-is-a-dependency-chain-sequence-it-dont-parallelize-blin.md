# Post is a dependency chain — sequence it, don't parallelize blindly

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any film & video production deliverable where this question is in play.

## The rule
Editorial → VFX → color → sound → conform → deliver has hard dependencies; starting color before the edit locks is rework, and the delivery date is set by the critical path.

## Why
This is house opinion §3 #3. Getting it wrong is the most common way a film & video production analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
