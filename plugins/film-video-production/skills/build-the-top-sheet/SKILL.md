# Skill: Build the top-sheet budget

A number without contingency and a defensible build is a wish.

## Step 1 — Build the categories
Above-the-line, below-the-line, post, and contingency (§3 #1).

## Step 2 — Build below-the-line bottom-up
Crew/gear/locations as rate × time × risk (§3 #7).

## Step 3 — Size the contingency
Set contingency to the project's risk (§3 #4).

## Step 4 — Reconcile to the spec
Check the budget delivers the spec (§3 #6).

## Output
A top-sheet budget with a bottom-up build, sized contingency, and spec reconciliation.
