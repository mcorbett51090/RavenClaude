# Tableau best-practice docs

Named, citable rules for Tableau engagements — each file is one rule, grounded in this plugin's own [`knowledge/`](../knowledge/) decision trees and enforced by its [`agents/`](../agents/). Read and apply a doc as a whole.

For the cross-tool rule format and the marketplace-wide index, see [`docs/best-practices/_TEMPLATE.md`](../../../docs/best-practices/_TEMPLATE.md) and [`docs/best-practices/README.md`](../../../docs/best-practices/README.md). For the plugin's house opinions these rules sit inside, see [`../CLAUDE.md`](../CLAUDE.md) §3-§4.
