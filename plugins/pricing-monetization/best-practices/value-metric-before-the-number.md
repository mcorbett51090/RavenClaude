# Decide the value metric before the number

The value metric — *what you charge per* — is the highest-leverage decision in
pricing. A right metric makes expansion automatic; a wrong one caps the company
forever, no matter how well you tune the number. Settle the metric (value-aligned,
expands with customer success, predictable enough to budget) before debating price.

**Do:** run `value-metric-design` and the failure question ("if this is 10× more
valuable, does our revenue grow?") before quoting any price.

**Flag:** a debate about the price *number* when the value metric is still unsettled
or is a cost proxy.
