# Build a NOI-growth asset plan

You are running `/commercial-real-estate:build-a-noi-growth-asset-plan` for `$ARGUMENTS`. Run it the way the team's specialists would — applying the house opinions in [`../CLAUDE.md`](../CLAUDE.md) §3.

## Steps (traverse top-to-bottom; do not skip)
1. Set the NOI target — Anchor to the acquisition model's NOI path; every quarter has a number to hit.
2. Sequence the levers — Order lease rollovers, recovery/expense-stop improvements, and capex by NOI impact and dependency (§3 #7).
3. Decide lease-up vs strategic vacancy — For each rollover, compare a fast fill at a low NER to holding for better credit/term (§3 #5).
4. Instrument the variance — Track realized vs underwritten NOI quarterly; name each variance as a leasing, opex, or recovery story.
