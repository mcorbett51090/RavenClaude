# Underwriting model spec

Define the model before building it.

- **In-place NOI:** rent roll basis, trailing opex, recovery ratio.
- **Step-up bridge:** each assumption, source, probability.
- **Returns:** going-in cap, unlevered + levered IRR, equity multiple.
- **Debt:** sizing constraint (DSCR/LTV/debt-yield), maturity, take-out.
- **Sensitivity:** exit cap × rent growth × refi rate grid.
- **Down case:** step-ups removed, exit cap widened.
