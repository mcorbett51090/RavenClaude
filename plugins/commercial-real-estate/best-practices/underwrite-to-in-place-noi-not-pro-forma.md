# Underwrite to in-place NOI, not pro-forma

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any time a deal is being evaluated.

## The rule
Build the base case on contractual in-place income; layer every step-up as an explicit, sourced, probability-weighted assumption.

## Why
A deal that only works on the seller's pro-forma is a story. Anchoring on in-place income is what separates a return from a hope.

## How to apply
- Start NOI from the rent roll and trailing opex.
- Make each mark-to-market/lease-up a named line with a source.
- Run a down case with the step-ups removed.
