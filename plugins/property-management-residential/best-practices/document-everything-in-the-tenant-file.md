# Document everything in the tenant file

**Status:** Pattern
**Domain:** Operations / risk management
**Applies to:** `property-management-residential`

---

## Why this exists

In a residential property management dispute — security deposit deduction, eviction, discrimination
complaint, habitability claim — the property manager's defense is the tenant file. A verbal
agreement that isn't in writing didn't happen. A maintenance request that was handled but not
documented leaves no evidence of timely response. A lease violation notice that was sent but not
retained can't be produced for the court.

Documentation discipline is not bureaucracy. It is the operational practice that separates property
managers who win disputes from property managers who settle them.

## How to apply

The tenant file for every resident should contain (at minimum):

**Before tenancy:**
- Signed application (with date received)
- Signed authorization for credit/background check
- Screening criteria document (the written policy in effect at application date)
- Decision memo (approval / denial with criterion-by-criterion rationale)
- Adverse action notice if denied based on consumer report (FCRA)

**At move-in:**
- Signed lease and all addenda
- Signed move-in inspection form with photos
- Security deposit receipt
- Utility responsibility documentation

**During tenancy:**
- Maintenance request records (work order number, date received, date resolved)
- Lease violation notices (with proof of delivery)
- Inspection reports (mid-lease, renewal, any entry notices)
- Payment arrangement agreements (written, signed)
- Any written correspondence with the resident

**At move-out:**
- Move-out inspection form with photos (signed if resident is present; noted if not)
- Security deposit disposition letter (with itemization and invoices/estimates)
- Proof of timely delivery of disposition (certified mail / PM software timestamp)

**Retention:** retain the complete tenant file for the jurisdiction's statute of limitations for
landlord-tenant claims (typically 3–6 years after tenancy ends — verify jurisdiction).

**Do:**

- Log every phone call (date, time, summary) in the PM software notes on the tenant record.
- Send lease violation and notice communications via the PM software portal (creates a delivery
  timestamp) and keep a copy.
- Attach photos to move-in and move-out inspections in the PM software — not on a phone camera roll.
- Store copies of all notices in the tenant file immediately after issuance.

**Don't:**

- Rely on verbal agreements with residents — confirm in writing via the portal or email.
- Delete or overwrite inspection photos after a dispute arises.
- Mark a security deposit disposition as "sent" without retaining a copy of the letter and proof
  of delivery.
- Skip logging contact attempts in the delinquency action ladder — the attempt log is the
  eviction file.

## Edge cases / when the rule does NOT apply

- **Routine maintenance completions with no dispute:** it is proportionate to retain a summary record
  (work order ID, completion date, cost) in the tenant file without saving every photo and invoice
  from a routine repair. However, for Emergency and Urgent items — especially habitability-related —
  retain the full documentation regardless.
