## Tier Definitions and SLAs

### Emergency — Habitability, Health, or Safety

**Response SLA:** ≤ \_\_\_ hour(s) acknowledgment to resident  
**Resolution SLA:** ≤ \_\_\_ hour(s) (recommended: ≤24h)  
**After-hours coverage:** Yes / No — contact: \_\_\_\_\_\_\_\_\_\_

**Qualifying conditions (examples — adapt for property type):**

| Category | Examples |
| --- | --- |
| HVAC — heat | No heat when outdoor temp ≤ \_\_°F |
| HVAC — cooling | No AC when outdoor temp ≥ \_\_°F (check jurisdiction requirement) |
| Plumbing | Sewage backup; toilet non-functional (only toilet in unit); active uncontrolled water leak |
| Electrical | Sparking; burning smell; outlet fire risk; full power outage |
| Gas | Gas smell; suspected leak |
| Security | Broken exterior door lock; broken window (ground floor); break-in damage |
| Structural | Ceiling collapse; major water intrusion |
| Pest / mold | Active infestation affecting habitability; visible active mold growth |
| Life safety | Smoke detector failure; CO detector alarm; fire damage |

### Urgent — Significant Comfort or Property Impact

**Response SLA:** ≤ \_\_\_ hour(s) acknowledgment  
**Resolution SLA:** ≤ \_\_\_ hour(s) (recommended: ≤72h)

**Qualifying conditions (examples):**

| Category | Examples |
| --- | --- |
| HVAC | System degraded but functioning; thermostat malfunction |
| Plumbing | Slow drain; running toilet; non-functional dishwasher; dripping faucet with water waste |
| Appliances | Refrigerator not cooling (risk to food); stove burner not working |
| Electrical | Single outlet not working; light fixture failure |
| Security | Interior lock malfunction |
| Water intrusion | Minor leak (contained, not active flooding) |

### Routine — Non-Urgent Repairs

**Response SLA:** ≤ \_\_\_ hour(s) acknowledgment (recommended: ≤24h business hours)  
**Resolution SLA:** ≤ \_\_\_ calendar days (recommended: ≤7)

**Qualifying conditions (examples):**

| Category | Examples |
| --- | --- |
| Cosmetic | Scuffs, nail holes, paint touch-up |
| Appliances | Dishwasher malfunction; garbage disposal; non-critical |
| Plumbing | Loose handle; cosmetic drip |
| Doors / hardware | Cabinet hinge; closet door; interior hardware |
| Flooring | Loose carpet edge; squeaky floor |
| General | Screen repair; blind replacement; light bulb (common area) |

## Triage Decision Rules

When a work order is received, apply these questions in order:

1. Does this affect heat, cooling, running water, plumbing function, electrical safety, gas, or
   structural safety? → **Emergency**
2. Does this significantly affect the resident's ability to use the unit (appliance failure,
   contained leak, partial electrical)? → **Urgent**
3. Everything else → **Routine**
4. When in doubt, assign the **higher** tier. A habitability item in Routine creates legal exposure.

## Vendor Notification Protocol

| Tier | Notification method | Confirmation required | If no confirmation |
| --- | --- | --- | --- |
| Emergency | PM software + direct phone call | ETA confirmed ≤30 min | Call backup vendor; escalate to PM |
| Urgent | PM software + text/call | ETA confirmed ≤1 hour | Call backup vendor |
| Routine | PM software dispatch | Confirmation next business day | Follow up via PM software |

## Escalation Triggers

| Condition | Action |
| --- | --- |
| Emergency resolution SLA missed | PM escalates to owner; contact backup vendor; flag to pm-compliance-advisor if habitability |
| Urgent resolution SLA missed | PM reviews; escalate to supervisor; update resident |
| Routine > 14 days unresolved | Review queue; vendor follow-up |
| Same issue re-opens within 30 days | Flag as callback; vendor performance review |
| 3+ callbacks on one vendor in 30 days | Vendor performance review; consider removal from panel |
