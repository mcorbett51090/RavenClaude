## Phase 1 — Move-Out Inspection (Day 0–1 after key surrender)

**Inspector:** \_\_\_\_\_\_\_\_\_\_\_\_\_ **Date:** \_\_\_\_\_\_\_\_\_

Compare current condition to the signed **move-in inspection form**. Check each item.

### Kitchen

- [ ] Appliances (stove, oven, refrigerator, dishwasher, microwave) — condition and function
- [ ] Cabinets and drawers — hardware, finish, damage
- [ ] Countertops — chips, burns, stains beyond normal wear
- [ ] Sink and faucet — function, seals, finish
- [ ] Garbage disposal — function
- [ ] Flooring — damage, cleaning needed
- [ ] Exhaust fan — function, cleaning needed

### Bathrooms (repeat for each)

- [ ] Toilet — function, seals, running
- [ ] Tub / shower — grout, caulk, finish, drain
- [ ] Vanity and sink — finish, hardware, drains
- [ ] Mirror — condition
- [ ] Exhaust fan — function
- [ ] Flooring — damage, cleaning

### Living / Dining

- [ ] Walls and ceilings — holes, stains, scuffs (normal wear vs. damage)
- [ ] Doors and frames — function, hardware, damage
- [ ] Windows and screens — function, condition, locks
- [ ] Blinds / window treatments — condition
- [ ] Flooring — type, damage, cleaning needed (carpet: stains, pet damage)
- [ ] Closets — shelving, doors, condition

### Bedrooms (repeat for each)

- [ ] Walls and ceilings — holes, stains
- [ ] Flooring — damage, cleaning
- [ ] Closets — doors, shelving
- [ ] Windows — function, condition, locks

### General / Systems

- [ ] HVAC — filter, thermostat function, supply/return grilles
- [ ] Water heater — no visible leaks, age noted
- [ ] Smoke detectors — quantity, function, battery date
- [ ] CO detectors — quantity, function, battery date
- [ ] Electrical outlets and switches — function, cover plates
- [ ] Exterior door locks and deadbolts — rekey needed? \_\_\_
- [ ] Patio / balcony / yard — condition, debris
- [ ] Garage / parking — damage, function
