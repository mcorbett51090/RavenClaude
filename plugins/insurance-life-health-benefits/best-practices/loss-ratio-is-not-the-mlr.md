# The loss ratio is not the ACA MLR test

The incurred loss ratio (claims ÷ premium) is the underwriting read on a group; the ACA medical-loss-ratio (MLR) is a defined regulatory test with rebate thresholds (80% individual/small-group, 85% large-group `[verify-at-build]`). They are not the same number and serve different purposes — never conflate them. A loss ratio below the MLR floor is a possible over-rating / rebate signal, which is information for the next negotiation, not a victory. Re-verify the current-year MLR thresholds before quoting them.
