# Skill: Optimize input economics

The last bushel often costs more than it returns.

## Step 1 — Build the response curve
Yield response to the input from data/trials (§3 #1).

## Step 2 — Find the economic optimum
Where marginal return = marginal cost at current prices.

## Step 3 — Vary by zone
Apply the optimum by management zone, not field-flat (§3 #2).

## Step 4 — Quantify the ROI
State the return vs an over/under-applied baseline.

## Output
An economic-optimum rate by zone with the input ROI.
