# Cost and margin are per acre, by field — never whole-farm only

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any precision agriculture deliverable where this question is in play.

## The rule
A whole-farm average buries the field that's losing money; per-acre cost and margin by field is where the decisions live.

## Why
This is house opinion §3 #4. Getting it wrong is the most common way a precision agriculture analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
