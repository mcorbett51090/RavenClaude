# Crop protection is threshold-and-resistance management, not calendar spraying

**Status:** Pattern. **Constitution:** §3 #6, §4.

## Use when
Any precision agriculture deliverable where this question is in play — read, applied, and cited whole.

## The rule
Scout-and-threshold beats calendar applications on cost and resistance; a prophylactic spray with no pressure is cost without return.

## Why it matters
This is house opinion §3 #6, distilled into a citable rule. Operators act on these deliverables, so a framing error here is not academic — it sends real money, staffing, or risk decisions the wrong way. The rule is cheap to apply and expensive to skip.

## How to apply
- Apply this **before** reaching for a method — it sets the framing, not the conclusion.
- Make every number in the deliverable carry a definition, a window, and a baseline (§3 #1).
- Cite a source + date for any external figure, or mark it `[unverified — training knowledge]` / `[ESTIMATE]` (§3 #8).
- When this rule and another both apply, route to [`agronomy-engagement-lead`](../agents/agronomy-engagement-lead.md) to sequence them — overlapping signals usually mean two drivers at once.
- Close with a recommendation that has an owner, a date, and an expected metric movement.

## The anti-pattern this prevents
The §4 failure mode: acting as if "crop protection is threshold-and-resistance management, not calendar spraying" weren't true — the most common way a precision agriculture analysis quietly misleads the operator who acts on it. The plugin's advisory hook flags a deliverable that reads as if this rule were ignored.

## Worked example
_Situation:_ Scout-and-threshold beats calendar applications on cost and resistance; a prophylactic spray with no pressure is cost without return. _Wrong move:_ skipping straight to a fix without applying this lens — you treat a symptom as the cause and prescribe the expensive wrong remedy. _Right move:_ apply the rule first; the framing it forces usually changes which of the two likeliest causes you chase, and it always changes what evidence you ask for next.
