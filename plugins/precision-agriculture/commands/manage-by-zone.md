# Manage by zone

You are running `/precision-agriculture:manage-by-zone` for `$ARGUMENTS`. Run it the way the team's specialists would — applying the house opinions in [`../CLAUDE.md`](../CLAUDE.md) §3.

## Steps (traverse top-to-bottom; do not skip)
1. Delineate zones — Build management zones from yield, soil, and topography (§3 #2).
2. Read yield by zone — Compare zone yields, not the field mean.
3. Vary the inputs — Put inputs where they return; pull them where they don't.
4. Measure the lift — Track per-zone return vs uniform application.

## Output
Delineated zones, a per-zone yield read, a variable-rate plan, and the return lift. See [`../skills/manage-by-zone/SKILL.md`](../skills/manage-by-zone/SKILL.md).
