# Estimate to a fully-loaded labor rate, not a wage

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any skilled trades contracting deliverable where this question is in play.

## The rule
The billable labor rate must absorb wage, burden, vehicle, tools, insurance, and overhead, or every hour sold loses money. A quote built on the technician's hourly wage is underwater before it starts. [unverified — training knowledge]

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a skilled trades contracting analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
