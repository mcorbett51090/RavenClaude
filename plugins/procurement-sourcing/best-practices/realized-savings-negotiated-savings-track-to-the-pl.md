# Realized savings ≠ negotiated savings — track to the P&L

**Status:** Pattern. **Constitution:** §3, §4.

## Use when
Any procurement & sourcing deliverable where this question is in play.

## The rule
A negotiated rate that never hits the P&L (leakage, maverick spend, volume miss) is theater; measure realized savings against a baseline that finance recognizes.

## Why
This is house opinion §3 #3. Getting it wrong is the most common way a procurement & sourcing analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
