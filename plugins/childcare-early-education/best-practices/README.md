# childcare-early-education — best-practice docs

Named, citable rules for the `childcare-early-education` team's specialists. Each file is **one rule**. Advisory operations knowledge — not legal, licensing, or financial advice; ratio, group-size, and subsidy specifics are **state-specific and `[verify-at-use]`**; no child or family PII.
