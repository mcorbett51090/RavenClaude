# Loyalty is repeat economics, not a member count

Measure the loyalty program by repeat rate, direct-booking share, and customer-lifetime-value lift — never by enrolled-member count, a vanity metric that rises with a sign-up prompt while real value falls. A loyal guest is cheaper to win than an OTA acquisition; the program earns its keep only if enrolled guests come back more and book direct more. Member count growing while repeat rate is flat is a points liability, not loyalty.
