# Alert rule review checklist

For each alert:
- [ ] Pages a human only when action is required now
- [ ] Symptom-based (or a cause that reliably precedes user pain)
- [ ] Tied to an SLO burn-rate where applicable
- [ ] Multi-window (fast + slow) to suppress blips
- [ ] Links to a runbook with the human action
- [ ] Has fired & been acted on in the last 90 days (else: delete)
