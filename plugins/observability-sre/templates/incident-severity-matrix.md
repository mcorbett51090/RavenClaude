# Incident severity matrix

| Sev | Definition | Response | Comms cadence |
|---|---|---|---|
| SEV1 | Major user-facing outage / data loss / security | IC + ops + comms, all-hands | every 15 min |
| SEV2 | Significant degradation, SLO burning fast | IC + ops | every 30 min |
| SEV3 | Minor / single-feature, budget intact | on-call handles | on resolve |

**Roles:** IC (coordinate, decide), Ops (fix), Comms (update stakeholders). Lean toward declaring.
