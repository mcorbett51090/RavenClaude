# Protocol feasibility is set before the first patient — design for enrollment

**Status:** Absolute rule. **Constitution:** §3, §4.

## Use when
Any clinical trials deliverable where this question is in play.

## The rule
Over two-thirds of sites miss enrollment and up to 50% enroll one or no patients; restrictive eligibility criteria are the single biggest enrollment killer. Stress-test feasibility at design, not at rescue.

## Why
This is house opinion §3 #1. Getting it wrong is the most common way a clinical trials analysis misleads the operator who acts on it.

## How to apply
- Apply it before reaching for a method.
- Make the relevant number carry a definition, window, and baseline (§3 #1).
- Cite the source + date for any external figure, or mark it `[unverified — training knowledge]`.
