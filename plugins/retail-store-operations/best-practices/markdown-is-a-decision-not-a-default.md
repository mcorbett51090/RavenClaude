# Markdown is a decision, not a default

Mark down to clear aged, seasonal, or terminal inventory on a cadence triggered by sell-through and weeks-of-supply — not reflexively, and not too late. The first markdown is usually the cheapest: depth compounds with time, so a late markdown clears deeper and carries more weeks of cost. Trigger on the metrics, not the calendar alone. And before discounting, check whether slow sell-through is an assortment problem (wrong SKU) rather than a price problem — fixing the price of a structurally wrong SKU just subsidizes a mistake.
