# The store is a P&L, not a cost center

Every store recommendation lands on a line of the store P&L: sales, gross margin, labor %, shrink, or controllable expense. A tactic that can't be traced to a P&L line is unmeasurable, so it doesn't ship. Run the four walls as a margin engine — sales × margin minus controllable cost equals four-wall contribution — not as a budget to defend. Naming the line also names the size of the prize, which is how you sequence what to fix first.
