# Treat the JavaScript bundle as a budget

Measure what you ship, code-split by route, and lazy-load heavy or below-the-fold code. Every dependency and every kilobyte is a first-load and Core-Web-Vitals cost the user pays. Enforce a bundle-size or Lighthouse budget in CI so performance can't rot one unnoticed PR at a time. A 2MB bundle is a slow app you chose to build.
