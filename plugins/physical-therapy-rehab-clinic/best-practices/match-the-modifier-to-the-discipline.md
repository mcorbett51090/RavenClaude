# Match the modifier to the discipline

**Status:** Absolute rule
**Domain:** Billing / modifiers
**Applies to:** `physical-therapy-rehab-clinic`

> Advisory only — not coding advice. Modifier edits are `[verify-at-use]` per payor.

---

## Why this exists

Therapy modifiers carry specific meaning. GP marks a PT plan of care (OT → GO, SLP → GN); KX attests documented medical necessity above the therapy threshold; 59 / X{EPSU} marks a genuinely distinct service against an NCCI edit. A mismatched or reflexive modifier is a denial — or, worse, a compliance exposure.

## How to apply

- **GP/GO/GN** by discipline of the plan of care.
- **KX** only when above the threshold *and* the documentation supports continued necessity.
- **59** only for a genuinely distinct service the note justifies — never to force an edit bypass.

## Edge cases / when the rule does NOT apply

PTA/OTA differentials (CO/CQ) and payor-specific edits vary; `[verify-at-use]`.
