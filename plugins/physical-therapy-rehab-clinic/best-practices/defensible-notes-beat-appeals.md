# Defensible notes beat appeals

**Status:** Pattern
**Domain:** Documentation / revenue cycle
**Applies to:** `physical-therapy-rehab-clinic`

> Advisory only. Appeal windows and documentation standards are `[verify-at-use]` per payor.

---

## Why this exists

An appeal is the expensive, low-yield recovery of a denial. The note written at the point of care is far cheaper than the appeal letter written months later — and it is the *evidence* the appeal would have to cite anyway. Documentation quality is the highest-leverage denial-prevention lever a clinic owns.

## How to apply

- Invest in the note at the visit: medical necessity, skilled care, the POC goal, functional change.
- When a denial does land, **appeal with the documentation that already exists** — lead with the note, not boilerplate.
- Feed denial root causes back into the documentation workflow.

## Edge cases / when the rule does NOT apply

Eligibility/authorization denials are front-end fixes, not documentation ones — route accordingly.
