# Skilled care must be skilled in the note

**Status:** Absolute rule
**Domain:** Documentation / compliance
**Applies to:** `physical-therapy-rehab-clinic`

> Advisory only. Documentation standards are `[verify-at-use]`.

---

## Why this exists

Reimbursement for therapy hinges on **skilled** care — work that requires the knowledge and judgment of a licensed therapist. If the note doesn't read as skilled, a reviewer concludes the service could have been delivered by an aide or self-directed, and denies it.

## How to apply

- Name the **clinical decision-making**: the progression/regression, the cueing or manual technique, the parameter adjustment and *why*.
- Record the patient's response and what it changed in the plan.
- Replace boilerplate ("tolerated well") with the skilled judgment exercised.

## Edge cases / when the rule does NOT apply

Maintenance therapy and some settings have distinct skilled-care standards; `[verify-at-use]`.
