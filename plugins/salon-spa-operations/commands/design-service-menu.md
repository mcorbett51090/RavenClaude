## Steps
1. Restructure each service family into good-better-best tiers with add-ons.
2. Cost each service (stylist time + product/back-bar + chair overhead) and price to a target margin — not to a competitor.
3. Map the service mix to margin per chair-hour; name the carriers and the loss-leaders.
4. If a price increase is in scope, traverse the price-increase tree in `knowledge/salon-spa-operations-decision-trees.md` and plan it (schedule, value message, segmentation, communicate to the book first).
5. Set a retail-to-service ratio target and the in-service recommend.
6. Emit using `templates/service-menu-and-pricing.md` + the Structured Output block.
