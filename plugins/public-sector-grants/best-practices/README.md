# Public-sector-grants best-practices

Atomic, enforceable rules the public-sector-grants agents apply. Each file is one rule with a short rationale; the agents cite them by filename. Canonical decision logic lives in [`../knowledge/public-sector-grants-decision-trees.md`](../knowledge/public-sector-grants-decision-trees.md); these rules are the always-on priors.

| Rule | Gist |
|---|---|
| fund-the-mission-not-the-money | Fit before fundability; an off-mission win is a loss |
| go-no-go-is-a-real-decision | Cost, probability, strings, sustainability — before writing |
| answer-the-funder-in-the-funders-words | Map the narrative to the NOFO's review criteria |
| logic-model-is-the-spine | Every dollar and claim traces to the logic model |
| objectives-are-smart-or-wishes | Specific, measurable, achievable, relevant, time-bound |
| budget-is-a-narrative-in-numbers | Every line justified, allowable, allocable, reasonable |
| allowable-allocable-reasonable-all-three | The 2 CFR 200 test for every federal dollar |
| cite-the-authority-not-a-memory | Thresholds/deadlines trace to current 2 CFR + award terms |
