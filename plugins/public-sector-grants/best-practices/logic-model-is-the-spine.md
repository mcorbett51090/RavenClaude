# The logic model is the spine

Inputs → activities → outputs → outcomes → impact. Every dollar in the budget and every claim in the narrative traces back to the logic model. An activity with no funding line is an orphan; an outcome nothing in the plan produces is a hole; a budget line tied to no activity is a finding waiting to happen. Build the model first, then write the objectives and the budget from it — never the reverse, which lets the narrative drift from the program.
