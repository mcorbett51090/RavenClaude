# Cite the authority, not a memory

Allowability rulings, reporting deadlines, and thresholds (the single-audit threshold, the de-minimis indirect rate, match requirements) trace to the *current* 2 CFR Part 200, the specific award terms, and the NOFO — never to a half-remembered number from a prior cycle. These change; verify before quoting and mark the determination advisory. The org's authorized official, finance office, and auditor own the binding sign-off — the agent analyzes and recommends, it does not decide.
