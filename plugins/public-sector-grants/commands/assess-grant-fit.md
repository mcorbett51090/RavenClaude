You are running `/public-sector-grants:assess-grant-fit`. Use `grant-strategist` + the `opportunity-fit-and-logic-model` skill.

## Steps
1. Screen for fit: eligibility (entity type, geography, population), mission alignment, capacity to deliver AND comply. If the screen fails, recommend no-go and stop.
2. Research the funder: stated priorities, recent awards, and the NOFO/RFP review/scoring criteria. Map the org's concept to each weighted criterion.
3. Build the logic model — inputs → activities → outputs → outcomes → impact — with the theory-of-change narrative; flag any orphan activity or unfunded outcome.
4. Make the go/no-go: name the cost-to-apply, the probability of award (as a range), the strings (match/cost-share, restrictions, reporting), and the sustainability tail past the period of performance.
5. Route: the narrative + budget → proposal-writer; match/allowability concerns → grants-compliance-analyst; donor-funded match/sustainability → nonprofit-fundraising.
6. Emit the recommendation + the Structured Output block (with `Funder requirement traced:` and `Handoff:`).
