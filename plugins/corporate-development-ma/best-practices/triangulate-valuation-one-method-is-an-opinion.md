# Triangulate valuation — one method is an opinion (§3 #4)

**Rule.** Cross DCF, trading comparables, and precedent transactions. When they diverge, the
divergence *is* the finding — explain it (growth, margin, control premium, cycle) rather than
averaging it into a false point estimate.

**Why.** Each method encodes different assumptions and blind spots. A single-method number is
an anchor someone chose; three methods bound the truth and expose where the assumptions fight.

**Do**
- Build all three and lay the ranges side by side.
- Explain every material divergence in the memo.
- Output a range and a walk-away, not a point.

**Don't**
- Present one method as "the" value.
- Average away a divergence you can't explain.
- Quote a current multiple from memory — cite source + date.
