# The thesis precedes the model (§3 #1)

**Rule.** State — in one sentence — why this target, why now, and how it creates value,
before building any valuation. If you can't, you don't have a deal; you have a spreadsheet
looking for a victim.

**Why.** A model built before a thesis rationalizes a number instead of testing a decision.
Every assumption bends toward the answer someone already wanted.

**Do**
- Write the one-sentence thesis first; name the value-creation drivers and the deal-breakers.
- Set a walk-away price before you start negotiating.
- Let the thesis, not a generic template, define the diligence plan.

**Don't**
- Open the model before the thesis is written.
- Confuse an accretive deal with a value-creating one.
