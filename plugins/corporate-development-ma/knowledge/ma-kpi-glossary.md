# M&A metric glossary

Definitions the team holds constant so every metric ships with a definition, window, and
baseline. Any implied benchmark is **directional** — cite a dated source before use (§3 #8).

| Metric | Definition | Notes / trap |
|---|---|---|
| **Enterprise value (EV)** | Equity value + net debt (+ minority interest, − associates). | The value of the operating business, capital-structure-neutral. |
| **EV / EBITDA** | EV ÷ EBITDA. The workhorse multiple. | Define EBITDA (reported vs adjusted); the add-backs are where value hides. |
| **EV / Revenue** | EV ÷ revenue. | For growth/pre-profit targets; pair with a path-to-margin. |
| **Control premium** | The % over unaffected trading price an acquirer pays for control. | Embedded in precedent transactions — don't double-count against DCF (§3 #4). |
| **Quality of earnings (QoE)** | How much of reported earnings is recurring, cash-backed, and sustainable. | The diligence read; separate one-time from recurring. Audited confirmation → accountants. |
| **Adjusted / normalized EBITDA** | EBITDA restated for one-time or owner-specific items. | Scrutinize every add-back; aggressive normalization inflates the multiple base. |
| **Net working capital (NWC) peg** | The target NWC level set at close; deviations adjust the price. | A common post-close dispute; define the peg mechanism up front. |
| **Synergies (run-rate)** | The steady-state annual benefit once fully captured. | Needs owner × date × cost-to-achieve (§3 #3); cost > revenue in confidence. |
| **Dis-synergies** | Value lost to attrition, disruption, and customer overlap post-close. | Real and routinely ignored; price pre-signing (§3 #6). |
| **Accretion / dilution** | The deal's effect on the acquirer's EPS. | A screen, not a thesis; accretive ≠ value-creating (§3 #1). |
| **IRR** | The discount rate at which the deal's cash flows net to zero. | Sensitive to exit assumptions; pair with MOIC. |
| **MOIC** | Multiple of invested capital (total value ÷ invested). | Cash-on-cash view; ignores time — read with IRR. |
| **Deal-breaker** | A finding that kills the deal regardless of price. | Named in the thesis up front; seeds diligence (§3 #5). |
| **Walk-away price** | The price/terms beyond which you decline. | A thesis with no walk-away is an intent to overpay. |

## Reporting rules (team standing)
- Every metric carries a **definition, window, and baseline**.
- Every multiple, comp, and market figure carries a **source + date** or a `[verify-at-use]` mark (§3 #8).
- Every synergy carries an **owner, a date, and a cost-to-achieve** (§3 #3).
- **No MNPI** the user isn't authorized to hold; this team gives no legal, tax, accounting, or fairness-opinion conclusions.
